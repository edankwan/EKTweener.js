This is an example of using the transform3d plugin. If you are not using the EKTweener.all.js, please make sure you added the transform3d plugin(EKTweenerTransform3dPlugin.js) into your html.



This plugin use getComputedStyle() to convert the style into Matrix2d/Matrix3d, which means you can freely use the standard transform functions including matrix3d(), translate3d(), translateX/Y/Z(), scale3d(), scaleX/Y/Z(), rotate3d() and rotateX/Y/Z(), and combine them into a string value.
