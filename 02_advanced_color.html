This is an example of using the color plugin. If you are not using the EKTweener.all.js, please make sure you added the color plugin(EKTweenerColorPlugin.js) into your html.



The color plugin support the color style using #RGB, #RRGGBB and rgb(R,G,B) formats. And the output will be in #RRGGBB format.

When you added EKTweener.all.js or EKTweenerColorPlugin.js to your document, it will automatically make the properties "color" and "backgroundColor" to use this plugin if the target is a DOM element. I didn't add the automatication to other style like borderColor because it may contain more than one color property.
