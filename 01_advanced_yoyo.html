This is an example of using the yoyo effect.



The property "yoyo" is an array which contains the properties that you want to use the yoyo effect.

As you might notice that if you click the button again before the previous transition is finished, the from value will become value at the moment you retrigger it. So if you want to make an effect that user may retrigger it during the transition, I recommend you to use fromTo() to make sure the from value stays the same as you expected.
