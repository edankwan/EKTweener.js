This is an example of using the basic EKTweener.to() transition.

EKTweener.to(element, duration, {properties});

  1. element can be the dom element or just an object
  2. duration is the duration of the transition in second
  3. properties are the properties of the element you want to animate. There are some preserved properties names for special features in EKTweener.

Other examples:

EKTweener.to(img, 1, {left: 100, top: 100});
EKTweener.to(img, 1, {width: 50});
