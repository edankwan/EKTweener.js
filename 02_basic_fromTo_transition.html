CLICK HERE
This is an example of using the basic EKTweener.fromTo() transition.

EKTweener.fromTo(element, duration, {from properties}, {to properties});

  1. element can be the dom element or just an object
  2. duration is the duration of the transition in second
  3. from properties are the properties of the element you want to tween from
  4. to properties are the properties of the element you want to tween to
